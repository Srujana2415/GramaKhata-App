Grama-Khata App📒💰

📌 Overview -
Grama-Khata is a mobile-based digital ledger application developed to simplify and modernize financial transaction management for small businesses, shop owners, and rural communities. The application replaces traditional paper-based bookkeeping with a secure, efficient, and user-friendly digital solution.
The app allows users to record and manage daily financial transactions under two categories: Give (money given) and Take (money received). It also provides balance tracking, customer management, reminders, and transaction history for better financial organization.
This project was developed as part of the *MindMatrix VTU Internship Program – Android App Development using GenAI*.

✨ Features -
💰 Record “Give” and “Take” transactions
👥 Customer management system
📊 Automatic balance calculation
📖 Transaction history tracking
🔔 Payment reminders and notifications
🎨 Color-coded entries for easy identification
📱 Simple and user-friendly interface
🔒 Secure local data management
